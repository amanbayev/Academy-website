# AIFC Academy Website

Multilingual Next.js MVP for AIFC Academy.

## Stack

- Next.js App Router
- TypeScript
- Tailwind CSS
- next-intl
- Local CMS-ready content data

## Local Development

```powershell
npm install
npm run dev
```

Default locale route: `/ru`.

## Routes

- `/ru`, `/en`, `/kk`
- `/[locale]/programs`
- `/[locale]/programs/[slug]`
- `/[locale]/corporate-training`
- `/[locale]/higher-education`
- `/[locale]/events`
- `/[locale]/about`
- `/[locale]/impact`
- `/[locale]/news`
- `/[locale]/contacts`
